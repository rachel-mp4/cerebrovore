initial setup
1. install go
2. install npm
3. `cd frontend/cbv`
4. `npm install`

dev build
1. `cd frontend/cbv`
2. `npm run dev`
3. open another terminal
4. `go run ./cmd`

prod build
1. `cd frontend/cbv`
2. `npm run build`
3. `cd -`
4. `go run ./cmd -prod`
